## Findings (per attack vector)

1. **"Two quadratics are one operation" (§2) — partially lands (wording).** Gravity's `δ²/3` is the squared **residual** `‖x−Px‖²` (the rejected complement); Born's `|⟨ψ|φ⟩|²` is the squared **retained component** `‖Px‖²` (probability weight). Opposite legs of the same right triangle — distinct projection functionals, not one operation. The defensible weaker claim (both inner-product-quadratic; shared parent is the metric) survives and suffices for the switch logic.
2. **Same-value-space step (G1) — lands hard.** The program asserts a shared *operator A*, not a shared *value space*. One A can be projective on metric Hilbert *V* and obstruction-registering on a bare relational *V*. The strong horn (→ modified gravity) smuggles shared-V; without it the result is a disjunction (shared-V → modified gravity; not → GR survives, unification lost), not a lean.
3. **Born-fundamental-vs-overlay (G4) — lands decisively (the biggest error).** The draft's empirical lever — "the discriminating criterion uses the metric" — is the **stage-1** γ̇ object, whose session verdict was **re-description, predicts nothing new** (Attempt 4). The projective nature the trilemma needs is **stage-2** Born, which `born-rule.md` §6.6.2 explicitly does *not* derive and names as the still-open keystone. The argument wields a stage-1 re-description result against the stage-2 open question. TRT's own two-stage factoring (obstruction stage-1 + unexplained projective stage-2 overlay) supports a fundamentally-obstruction-registering A → GR survives, unification retained.
4. **False trichotomy (vector 4) — lands.** A fourth combination holds all three legs: A fundamentally obstruction-registering (→GR) + projective Born as stage-2 overlay + shared A. It is TRT's own published position. "Cannot hold all three" is valid only as a conditional (IF shared-V AND fundamental-projective-A).
5. **Over-reach / drift (vector 5) — confirmed.** "Leans modified gravity" is unwarranted; it re-weights an already-listed horn only under the unproven shared-V premise, using a re-description result as if discriminating. **Same over-reach pattern the roster caught in Attempt 3** — a re-description result repurposed as discriminating content, now across sectors.
